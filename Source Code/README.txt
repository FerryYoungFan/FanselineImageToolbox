To run this application, you need the following modules:
在运行此应用前，请确保安装了如下模块

Python: Ver 3.7.4
Numpy: Ver 1.19.0
OpenCV (cv2: opencv-python): Ver 4.2.0.34 *
PyQt5: Ver 5.15.0


*For macOS Users: Please install opencv-python-headless instead of opencv-python
*对于 macOS 用户：请选用 opencv-python-headless 而不是 opencv-python


For Linux Users: Please run in Terminal:
对于 Linux 用户：请在 Terminal 中运行：
pip3 install Numpy
pip3 install opencv-python
sudo apt-get install python3-pyqt5



Also make sure you have downloaded the "GUI" folder
也请确保您下载了"GUI"文件夹

Then just run "FanselineToolbox.py" and have fun!
之后请运行“FanselineToolbox.py”即可！
